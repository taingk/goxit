# GOXIT

### Add dependencies

```
make add
```

### Install client dependencies

```
make install
```

### Rebuild client dependencies

```
make rebuild
```

### Start server, client and database

```
make run
```

### Build project

```
make build
```

### Start builded project, client and database

```
make run-build
```

### Stop server and database

```
make down
```

### Ports

```
API       8080
Client    8081
Adminer   8082
Postgres  8083
```
